# Our.Community.Polls

Simple Poll Property Editor for Umbraco

## 17.0.0 Update
Updated to Umbraco 17 / .NET 10 - to use new backoffice UI.

- Drag-and-drop answer sorter that posts repeated form keys (perfect for ordered answers).
- Workspace overview showing all polls, with quick actions.
- Responses view summarizing vote counts per answer.
- Custom tree and entity action to manage polls.
- UI: Lit web components, UUI components, UmbElementMixin, UmbSorterController, umbConfirmModal.
- Backoffice extension points: SidebarApp, Tree repository, Workspace views, Entity actions.
